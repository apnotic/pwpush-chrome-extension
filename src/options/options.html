About Password Pusher, Apnotic, and this extension

This extension helps you quickly create secure, expiring pushes from your browser.

Password Pusher is an open source tool for sharing sensitive content with automatic expiration. Learn more at pwpush.com, read the docs at docs.pwpush.com, review Best Practices, or view the OSS project on GitHub.

Apnotic is the company behind Password Pusher and the managed hosted regions used by this extension. Learn more at apnotic.com.

This extension is the official browser companion for Password Pusher. See project updates and source at github.com/apnotic/pwpush-chrome-extension.

Tip: Add an API token (available under User → API Tokens) to enable additional features such as Audit Logging, Auto-Dispatch, File Uploads, and more.
